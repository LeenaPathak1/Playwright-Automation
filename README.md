- Test Automation framework built using Playwright using Page Object Model pattern. 
- It includes some tests which can be using Cucumber testrunner and some which can be run using Playwright testrunner.
- It includes Web and API tests for a test websites - https://rahulshettyacademy.com/loginpagePractise/ and https://rahulshettyacademy.com/client/#/auth/login

- To run regression tests, use the required script from scripts section in package.json using following command: 
  **npm run <script name>**
